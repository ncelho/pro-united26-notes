# pro-united26-notes

本项目用于归档和发布多个独立的 HTML 页面，作为通用笔记与页面存储仓库。

## 项目简介

本仓库不针对任何特定网站或域名，主要用于存放各类自包含的 HTML 文件。这些文件可以是笔记、工具页面、演示页面或其他独立网页内容，便于通过 GitHub Pages 直接访问和分享。

## 目录说明

- `docs/`：主要存放 HTML 页面文件，每个文件通常为自包含的完整页面。
- `assets/`：存放页面所需的样式、脚本、图片等资源文件。
- `README.md`：本文件，提供仓库的基本说明。
- 其他目录 / 文件：可根据需要增加，用于组织不同类型的页面内容。

## 页面归档说明

- 每个 HTML 文件应尽量独立，避免跨文件依赖。
- 页面文件名建议使用英文或拼音，便于 URL 直接访问。
- 如需引用外部资源（如 CDN 库），请注明来源或版本。
- 本仓库不保证所有页面长期可用，但会尽力维护。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面。
- 请确保页面内容合法、不侵犯他人权益。
- 不建议存放包含敏感信息、恶意代码或违反 GitHub 服务条款的内容。
- 如有问题或建议，请通过 Issues 提出。

## 使用方式

1. 克隆本仓库到本地。
2. 在 `docs/` 目录下添加或修改 HTML 文件。
3. 提交并推送至 `main` 分支。
4. 通过 GitHub Pages 访问对应页面（通常路径为 `https://<用户名>.github.io/pro-united26-notes/`）。
